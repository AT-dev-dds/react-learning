
# ⚛️ React Conditional Rendering

## 🧠 Overview
Conditional rendering allows displaying different UI based on conditions.

👉 UI changes automatically based on state or props.

---

## 🔥 Methods

### 1. If / Else
Used for complex conditions

### 2. Ternary Operator
```js
{condition ? <A /> : <B />}
````

### 3. Logical AND (&&)

```js
{condition && <Component />}
```

### 4. Switch Case

Used for multiple conditions

### 5. Lists (.map())

Render items conditionally

---

## ⚡ Key Points

* Ternary → most commonly used
* && → render only if true
* if/else → for complex logic

---

## 🎯 Use Cases

* Authentication UI
* Loading states
* Dynamic content rendering

---

## 🚀 Summary

Conditional rendering helps control what users see based on conditions.
